# NAME

**bb** - A Benchmark tool for Bash script

# SYNOPSYS

    # Run as script
    loop=1000 bb.bash "<bash code A>" "<bash code B>"

    # Run as function
    . bb.bash
    loop=1000 bb "<bash code A>" "<bash code B>"

    # Debug
    BB_DEBUG=1 bb "<bash code A>" "<bash code B>"

# DESCRIPTION

A Benchmark script for Bash.
